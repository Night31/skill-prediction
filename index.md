## Manual installation:

_Steps:_
1. Download skill prediction with link at left side
2. Unzip archive in `bin/node_modules`
3. Check files location. Must be smth like `bin/node_modules/skill-prediction-master/index.js` 
3. Start proxy
4. DONE!
